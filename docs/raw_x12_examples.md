# Raw X12 270/271 Examples

Optum Eligibility API supports submitting **raw X12 270** payloads instead of JSON.

Reference for how to send raw X12:  
https://developer.optum.com/eligibilityandclaims/reference/rawx12

You must understand:
- X12 270 = Eligibility Inquiry
- X12 271 = Eligibility Response

This file should contain sample raw X12 transactions if you choose to test it.
