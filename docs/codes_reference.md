# Codes Reference for Eligibility

This page explains common code systems used in Optum eligibility responses.

---

## 🧾 Service Type Codes

Service Type Codes are X12 values that categorize the type of healthcare service.

Examples include:
| Code | Meaning |
|------|---------|
| 30 | Emergency Room |
| 60 | Dental |
| 82 | Vision |

Full official list (X12 standard):  
https://x12.org/codes/service-type-codes
